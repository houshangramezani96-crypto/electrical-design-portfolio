# LV Panel Design Checklist

## Input Data

- [ ] Load list received
- [ ] Voltage and frequency confirmed
- [ ] Installation environment confirmed
- [ ] Cable entry direction confirmed
- [ ] Protection philosophy confirmed
- [ ] Spare capacity required confirmed

## Electrical Checks

- [ ] Main incomer selected
- [ ] Short-circuit level considered
- [ ] Feeder protection selected
- [ ] Cable size coordinated with breaker
- [ ] Neutral and earth bars sized
- [ ] Control voltage selected
- [ ] Surge protection considered
- [ ] Metering requirements included

## Layout Checks

- [ ] Panel size sufficient
- [ ] Heat dissipation considered
- [ ] Cable bending radius considered
- [ ] Terminal space sufficient
- [ ] Labeling space included
- [ ] Maintenance access considered

## Documentation

- [ ] Single line diagram
- [ ] Panel layout
- [ ] Feeder schedule
- [ ] Terminal list
- [ ] BOM
- [ ] Test checklist
