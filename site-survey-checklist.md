# Site Survey Checklist

## General

- [ ] Project name confirmed
- [ ] Site address confirmed
- [ ] Responsible person identified
- [ ] Existing drawings collected
- [ ] Photos taken
- [ ] Safety restrictions noted

## Electrical Supply

- [ ] Incoming voltage checked
- [ ] Main panel inspected
- [ ] Transformer/generator/UPS status checked
- [ ] Load capacity reviewed
- [ ] Earthing system inspected

## Cable Routes

- [ ] Cable tray routes checked
- [ ] Conduit routes checked
- [ ] Wall/floor penetrations noted
- [ ] Mechanical obstacles noted
- [ ] Fire sealing requirements noted

## Panels

- [ ] Panel location suitable
- [ ] Access clearance sufficient
- [ ] Heat and ventilation considered
- [ ] Cable entry possible
- [ ] Labeling condition checked

## Lighting

- [ ] Room dimensions measured
- [ ] Ceiling height measured
- [ ] Existing lighting checked
- [ ] Required lux level identified
- [ ] Emergency lighting needs noted

## Deliverables After Survey

- Site report
- Photo log
- Measurement list
- Missing data list
- Preliminary BOM
