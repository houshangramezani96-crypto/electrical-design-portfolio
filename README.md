# Electrical Design Portfolio

Electrical engineering documentation samples for industrial and commercial projects.

**Focus areas:** LV panels, cable schedules, BOM/BOQ, earthing, lightning protection, DIALux lighting support, site survey checklists, and as-built documentation.

---

## About

This repository presents my electrical design and documentation approach for contractors, factories, and engineering teams.

The examples are sanitized and reusable. They show how I organize technical information, prepare deliverables, and support remote engineering work.

---

## Core Services

| Service | Deliverables |
|---|---|
| AutoCAD Electrical Redraw | Clean drawings, title block, tag structure, revision notes |
| Panel Documentation | Panel schedule, feeder list, protection notes, terminal list |
| BOQ / BOM | Quantity tables, material categories, specification notes |
| Earthing & Lightning | Earth rod schedule, conductor list, test checklist |
| DIALux Lighting | Room/area schedule, luminaire schedule, design assumptions |
| Site Survey | Field checklist, missing data log, risk notes |
