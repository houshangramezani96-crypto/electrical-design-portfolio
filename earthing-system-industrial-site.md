# Project Sample: Industrial Earthing System Documentation

## Scope

Prepare documentation for industrial site earthing and bonding.

## Inputs

- Site layout
- Soil data if available
- Electrical system type
- Equipment locations
- Required earth resistance target

## Deliverables

- Earthing material list
- Earth rod schedule
- Conductor routing notes
- Inspection pit schedule
- Test report template

## Engineering Checks

- Connection quality
- Corrosion protection
- Mechanical protection
- Accessibility for testing
- Bonding of metallic structures
