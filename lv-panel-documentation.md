# Project Sample: LV Panel Documentation

## Scope

Prepare clear documentation for a low-voltage electrical panel.

## Inputs

- Load list
- Panel photos
- Existing SLD or sketch
- Breaker ratings
- Cable sizes
- Control requirements

## Deliverables

- Single line diagram
- Feeder schedule
- Panel layout notes
- Cable list
- BOM
- Testing checklist

## Engineering Checks

- Breaker and cable coordination
- Panel space
- Neutral/earth bar sizing
- Labeling
- Control voltage
- Maintenance access
