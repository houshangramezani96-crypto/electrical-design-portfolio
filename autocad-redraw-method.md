# AutoCAD Electrical Redraw Method

## Goal

Convert unclear, old, hand-marked, scanned, or inconsistent electrical drawings into clean and maintainable engineering documents.

## Workflow

1. Review input drawings
2. Identify drawing type
   - SLD
   - Panel layout
   - Control diagram
   - Lighting layout
   - Power layout
   - Earthing layout
3. Define title block and revision format
4. Standardize layers
5. Standardize tags and equipment names
6. Redraw with clean dimensions and symbols
7. Add notes and missing information log
8. Submit PDF and DWG output

## Deliverables

- Clean DWG
- PDF version
- Revision notes
- Missing data / clarification list
- Optional BOM extraction

## Quality Checks

- [ ] Drawing scale checked
- [ ] Tags consistent
- [ ] Cable numbers consistent
- [ ] Symbols clear
- [ ] Notes readable
- [ ] Revision/date added
