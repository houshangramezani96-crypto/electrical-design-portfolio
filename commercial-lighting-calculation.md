# Project Sample: Commercial Lighting Calculation

## Scope

Prepare lighting calculation documentation for commercial or administrative areas.

## Inputs

- Floor plan
- Room dimensions
- Ceiling height
- Required illuminance
- Luminaire datasheets

## Deliverables

- DIALux report
- Luminaire schedule
- Lighting layout notes
- Emergency lighting assumptions
- Quantity table

## Engineering Checks

- Average illuminance
- Uniformity
- Glare consideration
- Maintenance factor
- Installation height
